3. Analysis modules¶

These modules are located in the MDAnalysis.analysis module (and the code lives in the MDAnalysis/analysis/ directory). They contain abstract functionality for specific analysis tasks (for example, bilayer leaflet selection). In many cases it will be preferable to use these provided tools over manually-coded alternatives because some components may have already been written in C for time-critical routines.

These modules are not imported by default; in order to use them one has to

import MDAnalysis.analysis
  • 3.1. Coordinate fitting and alignment — MDAnalysis.analysis.align
  • 3.2. Native contacts analysis — MDAnalysis.analysis.contacts
  • 3.3. Distance analysis — MDAnalysis.distances
  • 3.4. Generating densities from trajectories — MDAnalysis.analysis.density
  • 3.5. Leaflet identification — MDAnalysis.analysis.leaflet

Note

Some of the modules require additional Python packages such as scipy from the SciPy package or networkx from NetworkX. These package are not automatically installed (although one can add the [analysis] requirement to the easy_install command line to force their installation.
