4.5. DCD trajectory I/O — MDAnalysis.coordinates.DCD¶

Classes to read and write CHARMM/LAMMPS DCD binary trajectories. Trajectories can be read regardless of system-endianness as this is auto-detected.

The classes in this module are the reference implementations for the Trajectory API.
